Creates a copy of the env\+\_\+vars (entered as 3rd argument in main) on dynamic memory. 

The malloc-\/ed copy of environment variables passed as a third argument to the main is useful for builtin functions export and unset, which add/remove env vars. Clearer coding practise to have everything on dynamic memory, rather than having to keep track of what is on stack and what is dynamic/ needs to be freed
